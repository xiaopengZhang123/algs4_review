# 堆结构

## 建堆

```java
public void heapfiy(){
    //第一个非叶子结点开始下潜
    for(int i=size/2;i>=0;i--){
        down(i);
    }
    void down(int parent){
        //左孩子
        int left=parent*2;
        //右孩子
        int right=left+1;
        //意思是假设当前的最小值是parent结点
        int min=parent;
        if(left<size&&arr[parent]<arr[left]){
            min=left;
        }
        if(right<size&&arr[parent]<arr[right]){
            min=left;
        }
        if(min!=parent){
            swap(arr,min,parent);
            //parent是被交换了的结点
            //继续把这个位置的向下看，看用不用下潜
            down(min);
        }
    }
}

```
