# 队列的链表创建
1.由今天对队列的复习,让我发现了一种新的创建链表的方式
```
class Node{
    Node next;
    int val;
}
Node first,last;
void push(int item){
    Node old=last;
    //下面这一行相当于向后移动指针
    last=new Node();
    //设置last
    last.val=item;
    last.next=null;
    //只需要判断一下
    if(isEmpty()){
        first=last;
    }else{
        old.next=last;
    }
}

```

```
还要注意一下删除函数
int pop(){
    int val=first.val;
    first=first.next;
    if(isEmpty()){
        //最后空的话直接置空
        last=null;
    }
    return val;
}
```
