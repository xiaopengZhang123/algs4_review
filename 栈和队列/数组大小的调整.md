# 大小调整
1.扩张数组,这样可以不用一开始就去创建一个max数组,而删除也会减小空间
```
void resize(int length){
    int[]copy=new int[length];
    for(int i=0;i<arr.length;i++){
        copy[i]=arr[i];//arr是成员变量
    }
    arr=copy;//使数组变大
}

```

```
void push(int item){

```

3.pop函数
```
void pop(){
    if(size>0&&size==arr.length/4){
        resize(arr.length/2);
    }
    arr[size--];
}

```

经过调整,可以使数组使用量在25%-75%之间,扩张次数和缩减次数合理
