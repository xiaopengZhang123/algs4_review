# 数组模拟双向链表
其实核心就是边界情况的讨论
```java
构造函数,双端队列最大为 k 
class MyCircularDeque {
    private int[]arr;
    int size;
    private int first,last,max;
    public MyCircularDeque(int k) {
        arr=new int[k];
        size=0;
        first=last=0;
        max=k;
    }
    public boolean insertFront(int value) {
        if(isFull()){
            return false;
        }
        if(isEmpty()){
            first=last=0;
            arr[first]=value;
        }else{
            first=first==0?max-1:(first-1);
            arr[first]=value;
        }
        size++;
        return true;
    }
    
    public boolean insertLast(int value) {
        if(isFull()){
            return false;
        }
        if(isEmpty()){
            first=last=0;
            arr[last]=value;
        }else{
            last=last==max-1?0:last+1;
            arr[last]=value;
        }
        size++;
        return true;
    }
    
    public boolean deleteFront() {
        if(isEmpty()){
            return false;
        }
        first=first==max-1?0:first+1;
        size--;
        return true;
    }
    
    public boolean deleteLast() {
        if(isEmpty()){
            return false;
        }
        last=last==0?max-1:last-1;
        size--;
        return true;
    }
    
    public int getFront() {
        if(isEmpty()){
            return -1;
        }
        return arr[first];
    }
    
    public int getRear() {
        if(isEmpty()){
            return -1;
        }
        return arr[last];
    }
    
    public boolean isEmpty() {
        return size==0;
    }
    
    public boolean isFull() {
        return size==max;
    }
}

```

主要就是对数组的复用，给定一个长度为k的数组，像头部插入的话，就要判断一下下标为0时，就得向k-1位置插入，同样向尾部插入的话，就得考虑一下尾指针在index为k-1的情况，就得向0位置插入
